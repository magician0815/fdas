## 二、数据库设计变更

### 2.1 forex_daily表新增字段

```sql
-- 新增成交量字段
ALTER TABLE forex_daily ADD COLUMN volume BIGINT DEFAULT 0;
COMMENT ON COLUMN forex_daily.volume IS '成交量（外汇数据通常为0）';
```

### 2.2 新建用户图表配置表

```sql
-- 用户图表配置表（存储画线工具设置等）
CREATE TABLE user_chart_settings (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    setting_type VARCHAR(50) NOT NULL,       -- 配置类型：drawing_tools/theme/indicators
    setting_key VARCHAR(100) NOT NULL,       -- 配置键名
    setting_value JSONB NOT NULL,            -- 配置值（JSON格式）
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(user_id, setting_type, setting_key)
);

COMMENT ON TABLE user_chart_settings IS '用户图表个性化配置存储表';
COMMENT ON COLUMN user_chart_settings.id IS '配置记录唯一标识ID';
COMMENT ON COLUMN user_chart_settings.user_id IS '关联用户ID';
COMMENT ON COLUMN user_chart_settings.setting_type IS '配置类型（drawing_tools/theme/indicators/view）';
COMMENT ON COLUMN user_chart_settings.setting_key IS '配置键名';
COMMENT ON COLUMN user_chart_settings.setting_value IS '配置值（JSON格式）';
COMMENT ON COLUMN user_chart_settings.created_at IS '创建时间';
COMMENT ON COLUMN user_chart_settings.updated_at IS '更新时间';

CREATE INDEX idx_user_chart_settings_user ON user_chart_settings(user_id);
CREATE INDEX idx_user_chart_settings_type ON user_chart_settings(setting_type);
```

## 四、P0级功能详细描述（12项）

### F001 - K线基础渲染（蜡烛图）

**功能描述**: 实现标准蜡烛图渲染，红涨绿跌配色，支持OHLC四个价格点渲染

**验收标准**:
- [ ] 蜡烛图正确渲染OHLC数据
- [ ] 红色表示上涨，绿色表示下跌
- [ ] 开盘价、收盘价、最高价、最低价正确显示
- [ ] 价格精度：外汇4位小数

**预估工作量**: 1天

**依赖关系**: 无依赖，可独立开发

---

### F002 - K线形态切换（蜡烛图/折线图）

**功能描述**: 支持蜡烛图与收盘价折线图一键切换，切换时无闪屏

**验收标准**:
- [ ] 一键切换按钮，切换动画平滑
- [ ] 切换后数据范围保持一致
- [ ] 记住用户上次选择的形态

**预估工作量**: 0.5天

**依赖关系**: 依赖F001

---

### F003 - 价格坐标轴系统

- [ ] 右侧价格轴可选显示
- [ ] 缩放时价格轴自动适配
- [ ] 价格精度：外汇4位小数

**预估工作量**: 0.5天

**依赖关系**: 依赖F001

## 十一、验收标准

### 全局验收标准

1. **性能标准**
   - 首屏加载 < 2秒
   - 缩放/平移响应 < 100ms
   - 无明显卡顿、闪屏

2. **渲染标准**
   - OHLC数据精度正确
   - 时间轴对齐准确
   - 主副图同步无错位
   - 抗锯齿渲染清晰

3. **交互标准**
   - 鼠标/键盘操作响应准确
   - 光标定位精度 < 1像素偏差

4. **兼容标准**
   - Chrome/Firefox/Safari最新版兼容
